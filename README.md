# MoviePilot-Plugins

[![Release Preflight](https://github.com/liuyuexi1987/MoviePilot-Plugins/actions/workflows/ci.yml/badge.svg)](https://github.com/liuyuexi1987/MoviePilot-Plugins/actions/workflows/ci.yml)

两个插件，解决两个问题：

- **Agent影视助手**：面向外部智能体的 MoviePilot 稳定控制层，同时内置飞书桥接入口；用插件 + skill/helper 把资源搜索、转存、下载、订阅、签到和修复收进固定命令流，减少智能体乱调用和长上下文 Token 消耗
- **AI识别增强**：文件名识别失败时用 LLM 兜底，并沉淀失败样本与识别词建议

它的重点不是再做一个聊天机器人，而是让 `OpenClaw`、`Hermes`、`WorkBuddy` 这类外部智能体用低 Token、可续接、可确认的方式稳定控制 MoviePilot。资源流由插件执行，智能体只负责理解需求和展示结果。

如果你是第一次接触这个仓库，先看这两个插件就够了。

---

## 第一步：安装插件

MoviePilot -> 插件市场 -> 添加仓库：

```text
https://github.com/liuyuexi1987/MoviePilot-Plugins
```

然后优先安装：

```text
Agent影视助手
AI识别增强
```

其中 `Agent影视助手` 是外部智能体接入 MoviePilot 的主入口。它让智能体不要自己乱拼影巢、盘搜、115、夸克接口，而是用固定命令把资源流统一交给插件执行。

---

## 第二步：接入外部智能体

如果你要让 `OpenClaw`、`Hermes`、`WorkBuddy` 这类外部智能体控制 MoviePilot，装完插件后先看这里，详细步骤一定以文档为准：

[外部智能体接入 Agent影视助手](./docs/AGENT_RESOURCE_OFFICER_EXTERNAL_AGENTS.md)

如果长时间使用同一个会话后，智能体开始误解编号或套用旧规则，可以对它说 `校准影视技能`；智能体会先检查并拉取 GitHub 仓库最新版，再重新加载影视技能规则。

---

## 第三步：填配置

打开 `Agent影视助手` 设置页面，按你要用的功能填写：

| 你想用的功能 | 需要填什么 |
|---|---|
| 盘搜搜索 | `盘搜 API 地址` |
| 影巢搜索 | `影巢 OpenAPI Key` |
| 115 转存 | `115 默认目录`，推荐搭配 `P115StrmHelper`；没有它也可以直接走 `115登录` 扫码 |
| 夸克转存 | `夸克 Cookie` 或 `CookieCloud` |
| 飞书入口 | 飞书应用的 `App ID` / `App Secret` |
| PT 下载 | 一般只要 MoviePilot 原生下载器正常即可；如果 MP 和 qB 不在一台机器，可额外填写 `PT 下载保存路径` |

不用的功能可以不填，插件会自动跳过。

这里重点说一下 Cookie：

夸克优先用 `CookieCloud` 导入，影巢可以先尝试填写用户名、密码让插件自动登录。如果你没有 `CookieCloud`，或者遇到夸克转存失败、影巢签到 Cookie 失效、自动登录拿到的 Cookie 不适合签到，不建议新手手动去浏览器里一点点找。接了外部智能体后，可以直接对智能体说：

```text
刷新夸克Cookie
刷新影巢Cookie
修复夸克转存
修复影巢签到
```

前提是智能体所在电脑的浏览器里已经登录过夸克/影巢，并且还能读到有效 Cookie。智能体会按 skill/helper 里的规则调用仓库内置的 Cookie 导出工具，从智能体电脑浏览器读取登录态，再写回 MoviePilot。

## 第四步：直接开始用

装好、填好配置后，直接在外部智能体里发这些命令，例如 `OpenClaw`、`Hermes`、`WorkBuddy` 等：

### 搜索

```text
MP搜索 流浪地球2
搜索 流浪地球2
影巢搜索 流浪地球2
盘搜搜索 流浪地球2
```

- `MP搜索` / `PT搜索`：走 MoviePilot 原生 PT 搜索
- `搜索`：默认优先走 MP/PT
- `盘搜搜索`：先查盘搜；如果盘搜没结果，会按开关补查影巢
- `影巢搜索`：先查影巢；如果影巢没结果，会按开关补查盘搜
- `云盘搜索`：已废弃，收到后只会提示改用 `盘搜搜索` / `影巢搜索`
- `搜索 第 3 集`、`搜索 E03` 这类带集数线索的写法，会直接按 MP/PT 搜索，不再回退到云盘
- 云盘侧更新检查请显式使用 `盘搜更新检查 <片名>` / `影巢更新检查 <片名>`
- `MP/PT` 没结果时，只给下一步建议，不会自动跳去盘搜或影巢

### 转存 / 下载

```text
转存 流浪地球2
115转存 流浪地球2
夸克转存 流浪地球2
下载 流浪地球2
```

- `转存 <片名>` 默认等同 `115转存 <片名>`
- 只有明确发送 `夸克转存 <片名>` 才会走夸克
- `下载 <片名>` 走 MoviePilot 原生 PT 下载链，先找片、再列出 PT 候选

### 选择与翻页

```text
1
下载1
选择 1 详情
n
```

- `1`：PT 结果里直接下载；云盘结果里继续转存/解锁
- `下载1`：给第 1 条 PT 结果生成下载计划，不会立刻下载
- `选择 1 详情` / `详情 1`：只用于云盘/影巢详情
- `n`：下一页

### 其他常用 / 进阶

```text
订阅 流浪地球2
更新检查 流浪地球2
115登录
115状态
影巢签到
影巢签到日志
修复影巢签到
最佳片源
计划选择 1
执行计划
校准影视技能
```

- `订阅 <片名>`：走 MoviePilot 订阅链。
- `更新检查 <片名>`：对比 TMDB 进度、盘搜和影巢资源，判断是否有新集。
- `115登录` / `115状态`：扫码登录 115，并检查当前登录是否可用。
- `影巢签到`：执行影巢签到；如果失败，可先看 `影巢签到日志`，再让智能体执行 `修复影巢签到`。
- `最佳片源`：在当前结果里只读查看插件评分最高的候选，不会直接转存或下载。
- `计划选择 1`：为第 1 条结果生成待确认计划，适合你想先确认再执行的场景。
- `执行计划`：确认执行当前待确认计划。
- `校准影视技能`：长会话用久后，如果智能体开始误解编号、套用旧格式或把详情当执行，就用它先检查/拉取 GitHub 仓库最新版，再重新加载规则。

搜索结果里经常会出现 `智能建议`，它不是让智能体自由发挥，而是插件根据画质、集数、字幕、体积、做种、来源和你的偏好做的选择建议。PT 结果按建议直接回编号下载；云盘/影巢想先确认再用 `选择 1 详情` 或 `详情 1`。

完整命令见：[全部命令](./docs/ALL_COMMANDS.md)

## 飞书怎么用

如果不使用外部智能体，只想用命令功能，也可以只配置飞书入口。

配好飞书后，它就像 TG / 企业微信一样，可以作为 MoviePilot 的资源命令入口。你直接在飞书里发命令，插件会在 MoviePilot 里完成搜索、转存、下载、签到等操作。

飞书最快设置：

- 在 `Agent影视助手` 设置里填 `飞书 App ID` 和 `飞书 App Secret`
- 打开 `启用内置飞书入口`
- 先把 `允许所有飞书会话` 打开，其它项先保持默认
- 保存后，去飞书里发 `版本` 或 `菜单`，能收到回复就说明已经接通
- 后面如果要收紧范围，再回头填写 `Chat ID / Open ID` 白名单

| 飞书里发什么 | 会做什么 |
|---|---|
| `MP搜索 流浪地球2` | PT 搜索 |
| `盘搜搜索 流浪地球2` | 盘搜搜索 |
| `影巢搜索 流浪地球2` | 影巢搜索 |
| `下载资源 1` | 下载第 1 条 PT 结果 |
| `选择 1` | 继续处理第 1 条当前结果 |
| `帮助` | 查看帮助 |

飞书里还有一些快捷别名，但新手直接用完整命令最稳。

## AI识别增强是做什么的

MoviePilot 整理文件时，会先识别文件名里的片名、年份、季、集。

如果原生识别失败，`AI识别增强` 会：

1. 用当前 MoviePilot 配置好的 LLM 做一次结构化识别
2. 把识别结果交回 MoviePilot，继续走原生整理链
3. 保存失败样本，方便后续生成自定义识别词

它不会绕过 MoviePilot 原生整理流程，只是在识别环节加了一层兜底。

详细说明见：[AI识别增强](./AIRecognizerEnhancer/README.md)

## 旧插件还要不要用

下面这些老插件还在仓库里，但新装一般不再推荐优先用它们：

| 旧插件 | 主要用途 | 现在的建议 |
|---|---|---|
| `FeishuCommandBridgeLong` | 单独把飞书作为命令 Channel | 只想要飞书入口、不需要 Agent影视助手整套资源流时可以用 |
| `HdhiveOpenApi` | 影巢搜索、账号、签到等独立能力 | 只想单独使用影巢能力或签到时可以用 |
| `QuarkShareSaver` | 夸克分享链接转存 | 只想单独做夸克分享转存时可以用 |

如果你想把盘搜、影巢、115、夸克、PT 下载、飞书和外部智能体统一到一个入口，优先直接用 `Agent影视助手`。
