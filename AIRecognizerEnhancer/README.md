# AI识别增强

这是重构中的新识别插件目录，用来替代当前依赖外部 AI Gateway 的旧识别转发思路。

目标不是简单复制旧插件，而是重写成：

- 直接复用 MoviePilot 当前已启用的 LLM 配置
- 在原生识别失败后做本地结构化 AI 辅助识别
- 自动回到 MoviePilot 整理链路继续处理

## 计划承接的来源

- `AIRecoginzerForwarder`

## 首期目标

- 监听识别失败场景
- 调用 MP 内置 LLM 进行标题、年份、类型、季集等结构化判断
- 触发二次整理

## 二期方向

- 生成“自定义识别词”建议
- 识别失败样本沉淀
- 与 MP 原生 Agent / Skill 能力联动

## 迁移原则

- 不再把外部 Gateway 当作唯一依赖
- 尽量减少容器间回调和额外网络链路
- 保持插件职责只聚焦在“识别增强”

- 还未开始写正式插件代码
- 后续会在这里完成重构实现
