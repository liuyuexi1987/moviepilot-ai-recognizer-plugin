# FeishuCommandBridgeLong

飞书命令桥接插件的长连接版本，面向 MoviePilot 使用。

## 目标

- 不再依赖公网 webhook / HTTPS 回调地址
- 使用飞书长连接直接接收 `im.message.receive_v1`
- 将飞书文本命令映射成 MoviePilot 内部命令或 115 整理动作
- 对关键动作回执开始、完成、统计结果

## 当前支持

- 飞书长连接接收消息
- 智能单入口：自动识别夸克链接、115 链接、影巢搜索、盘搜搜索
- 群聊 / 用户白名单
- 中文别名映射
- 跨实例事件去重，避免同一 `event_id` 重复执行
- 可选回执消息
- 115 手动整理默认目录批量执行
- STRM 增量 / 全量命令桥接
- PT 资源搜索结果回执（前 10 条）
- 按序号直接下发下载任务
- 直接添加订阅
- 添加订阅后立即触发一次搜索

## 依赖

```txt
lark-oapi==1.5.3
```

## 可选增强

- 可在插件配置中填写 `TMDB API Key（可选）`
- 作用：用于 `详情` / `审查` 时按需补充当前候选页的 1 到 2 个主演名
- 不填写时，影巢搜索与解锁流程仍可正常使用，只是详情模式不显示主演

## 执行后端

- 插件现在支持三种执行后端：
  - `旧桥接直连`：继续直接调用旧链路，速度更快，适合保持当前飞书使用习惯
  - `自动优先新主线，失败回落旧桥接`：优先把影巢 / 115 / 夸克动作转交给 `Agent资源官`
  - `仅走 Agent资源官 新主线`：强制只走新主线，适合联调新插件
- 当前更推荐日常飞书使用先保留 `旧桥接直连`
- `0.5.16` 起，切到 `自动` 或 `仅走 Agent资源官` 后，飞书桥接的智能入口和继续选择都可以整条委托给 `Agent资源官`

## 飞书后台配置

在飞书开放平台中：

1. 打开应用
2. 开启事件订阅
3. 订阅方式改为“长连接”
4. 勾选事件：`im.message.receive_v1`
5. 填写应用的 `App ID`、`App Secret`
6. `Verification Token` 可填写，但长连接模式本身不依赖公网 challenge

## 当前命令语义

### 基础命令

```txt
版本
```

对应：

```txt
/version
```

### 智能单入口

```txt
处理 流浪地球2
处理 https://pan.quark.cn/s/xxxx pwd=abcd path=/最新动画
处理 https://115cdn.com/s/xxxx?password=abcd path=/待整理
```

- 发送片名时：先返回候选影片列表；如果存在重名或不同年份版本，先按编号确认影片，再返回资源列表
- 发送夸克链接时：自动走 `QuarkShareSaver`
- 发送 115 分享链接时：自动走 `HdhiveOpenApi` 的 115 转存接口
- 可选参数：
  - `path=/目录`
  - `pwd=提取码`
  - `type=movie|tv`
  - `year=2024`

现在也支持直接发送裸链接，不需要再写 `处理` 前缀：

```txt
https://pan.quark.cn/s/xxxx pwd=abcd path=/最新动画
https://115cdn.com/s/xxxx?password=abcd path=/待整理
```

- 飞书收到后会自动识别链接类型
- 夸克链接直接走夸克转存
- 115 链接直接走 115 转存

```txt
选择 1
选择 1 path=/最新动画
```

- 对上一轮 `处理 片名` 的结果按编号继续
- 当前会自动执行：
  - 第一步：在影巢候选影片和资源列表之间继续流转
  - 第二步：选中最终资源后执行影巢解锁
  - 如返回 115 / 夸克资源则继续转存到指定目录
- 适合做“搜 -> 选 -> 落地”的轻流程

```txt
盘搜搜索 流浪地球2
```

- 走盘搜接口并按相关性回执结果
- 默认固定展示相关性最高的 115 前 6 条和夸克前 6 条
- 结果同样缓存在当前飞书会话里
- 也兼容简写：
  - `ps流浪地球2`
  - `1流浪地球2`
- 可以直接继续：

```txt
选择 1
选择 1 path=/最新动画
```

- 如果选中项是 115 或夸克链接，会直接转存
- 如果是其他网盘类型，会明确提示当前暂不支持直接落地

### 影巢搜索现在的两段式流程

```txt
影巢搜索 蜘蛛侠
2蜘蛛侠
yc蜘蛛侠
```

- 第 1 步：先展示候选影片，例如不同年份、不同系列版本；默认只展示片名、年份和类型，优先保证首屏速度
- 如果候选超过 10 个，会按每页 10 条展示；可直接回复 `n 下一页` 继续翻页
- 如果需要补充当前候选页全部主演，可回复 `详情` 或 `审查`
- `详情` / `审查` 只会补当前页候选，并发查询主演，尽量缩短等待时间
- 第 2 步：用户回复 `选择 1` 后，再展示该影片的影巢资源
- 资源列表默认按 `115 前 6 条 + 夸克前 6 条` 分区展示，和盘搜保持同样的阅读方式

### 115 整理命令

```txt
刮削
```

- 不带参数时：读取 `P115StrmHelper.pan_transfer_paths`
- 如果配置了多个待整理目录，会全部逐个执行

```txt
刮削 /待整理/
```

- 只执行指定目录

### STRM 命令

```txt
生成STRM
同步STRM
```

- 走 `/p115_inc_sync`
- 按 `P115StrmHelper.increment_sync_strm_paths` 中配置的全部媒体库做增量生成

```txt
全量STRM
```

- 走 `/p115_full_sync`
- 按 `P115StrmHelper.full_sync_strm_paths` 中配置的全部媒体库做全量生成

```txt
指定路径STRM /某路径
```

- 走 `/p115_strm /某路径`
- 只对指定路径做全量生成

### 搜索与下载命令

```txt
搜索资源 流浪地球2
搜索 流浪地球2
搜索流浪地球2
```

- 先由 MoviePilot 识别媒体信息
- 再走 MP 自己的 PT 搜索链返回前 10 条资源
- 结果会缓存在当前飞书会话里，方便下一步按序号下载
- 空格可选，以上三种写法都支持

```txt
下载资源 1
下载 1
下载1
```

- 下载上一条 `搜索资源` 返回列表中的第 1 条
- 适合飞书里快速完成“搜 -> 选 -> 下”
- 空格可选

### 订阅命令

```txt
订阅媒体 流浪地球2
订阅 流浪地球2
订阅流浪地球2
```

- 识别媒体后直接添加订阅

```txt
订阅并搜索 流浪地球2
订阅搜索 流浪地球2
订阅搜索流浪地球2
```

- 添加订阅
- 立刻触发一次订阅搜索
- 适合做“先建订阅，再尝试马上拉资源”的一条龙入口
- 空格可选

## 智能体 API

除了飞书消息入口，这个插件现在也提供两条更适合智能体调用的 API：

```txt
POST /api/v1/plugin/FeishuCommandBridgeLong/assistant/route
POST /api/v1/plugin/FeishuCommandBridgeLong/assistant/pick
```

`route` 示例：

```json
{
  "session": "workbuddy-demo",
  "text": "MP搜索 流浪地球2"
}
```

或：

```json
{
  "session": "workbuddy-demo",
  "text": "影巢搜索 流浪地球2"
}
```

或：

```json
{
  "session": "workbuddy-demo",
  "text": "盘搜搜索 流浪地球2"
}
```

或：

```json
{
  "session": "workbuddy-demo",
  "text": "https://pan.quark.cn/s/xxxx pwd=abcd path=/最新动画"
}
```

`pick` 示例：

```json
{
  "session": "workbuddy-demo",
  "index": 1
}
```

说明：

- `session` 用来隔离不同智能体或不同会话的选择缓存
- `route` 负责分流和返回候选
- `pick` 负责对上一次候选结果按编号继续执行
- 推荐约定：
  - `MP搜索 片名` 走 MoviePilot 原生 PT 搜索
  - `影巢搜索 片名` 走影巢搜索/解锁工作流
  - `盘搜搜索 片名` 走盘搜聚合直链搜索
  - 直接发送 `115` / `夸克` 链接则自动转存
- 这样飞书和智能体可以共用同一条稳定执行链路
- 如果飞书端切到 `auto` 或 `仅走 Agent资源官 新主线`，这两条 API 也会复用同一条新资源主线

## 搜索前缀

默认搜索仍然保持 MoviePilot 原生搜索。

```txt
搜索 流浪地球2
MP搜索 流浪地球2
```

- 这两种都走 MP 原生 PT 搜索
- 适合“搜种 -> 下载 -> 订阅”

```txt
影巢搜索 流浪地球2
```

- 明确走影巢关键词搜索
- 返回影巢资源结果
- 后续可接 `选择 1`
- 也兼容简写：
  - `yc流浪地球2`
  - `2流浪地球2`

```txt
盘搜搜索 流浪地球2
```

- 明确走本机 PanSou 服务
- 默认展示相关性最高的 115 前 6 条和夸克前 6 条
- 盘搜结果本身不直接下发到 MP 下载器
- 如果搜到的是夸克或 115 链接，可以把结果链接再发给 `处理 ...`

## 回执

对 `刮削` 命令会回：

- 开始回执
- 完成回执
- 总计 / 成功 / 失败 / 跳过统计
- 失败示例（最多 3 条）

## 已知限制

- 长连接线程启动后暂未实现无重启主动停止
- 修改 `app_id / app_secret` 后建议重启 MoviePilot
- 115 相关命令依赖 `P115StrmHelper`
